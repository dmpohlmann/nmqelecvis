# Electricity Cost Comparison

Interactive comparison of embedded network electricity costs (Altogether Group) vs retail plans for a Queensland body corporate scheme.

## Setup

```bash
npm install
npm run dev
```

## Deploy to Netlify

Connect this repo to Netlify with:
- **Build command:** `npm run build`
- **Publish directory:** `dist`

## Data

- Retail rates from Energy Made Easy (energymadeeasy.gov.au), Carseldine 4034, March 2026
- Consumption benchmarks from AER Residential Energy Consumption Benchmarks (2020), Climate Zone 2
